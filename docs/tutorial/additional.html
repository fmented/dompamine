Creating Element Using MultiElementObject

Creating element only works with Element and not with MultiElement.
Hovever, you can do this if you want
though it's not recommended
[...MultiElementObject].forEach(i=>{i.create.bottom("h1", "", {innerText: "some text"}))

Removing Element Using MultiElementObject

Removing element only works with Element and not with MultiElement.
Hovever, you can do this if you want
though again it's not really recommended
[...MultiElementObject].forEach(i=>{i.remove.self()})

Fire An Event Using MultiElementObject

Firing event only works with Element and not with MultiElement.
Hovever, you can do this if you want
though again it's not really recommended
[...MultiElementObject].forEach(i=>{i.click()})
